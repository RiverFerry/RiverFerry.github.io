总结

模板类的构造函数在c++17前是不能自动推导参数类型的，所以需要手动指定类型：

std::pair <int,int> p(1,2);

而c++17后可以直接：

std::pair p(1, 'a');

具体原因我也不是很懂，可以看参考的文章。但是普通函数是支持类型推导的，所以之前c++17以前实现了make_pair：

template <class T1,class T2>
pair<T1,T2> make_pair (T1 x, T2 y)
{
  return ( pair<T1,T2>(x,y) );
}
