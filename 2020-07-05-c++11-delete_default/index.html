注意点：

  • 只针对于这几种：默认构造函数，默认析构函数，拷贝/移动构造函数，赋值/移动赋值操作符重载函数。
  • 不可用于类外
  • default的函数由编译器自己实现函数体，性能比自己写的性能要高
