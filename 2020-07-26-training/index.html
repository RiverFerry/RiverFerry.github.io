002 数组中重复的数字

题目 来源 完成时间
找出数组中重复的数字 剑指offer 20200417

在一个长度为n的数组里的所有数字都在0 ~ n-1的范围内。数组中某些数字是重复的，但不知道有几个数字重复了，也不知道每个数字重复了几次。请找出数组中任意一个重复的数字。例如，如果输入长度为7的数组{2,3,1,0,2,5,3},那么对应的输出是数字2或者3

方法1：排序，排完序遍历数组

方法2：哈希表，插入判断是否重复

方法3：让数组索引值和数组的元素值相等，data[0] = 0, data[1] = 1,对应不上的不管

方法 时间复杂度 空间复杂度
排序 O(nlogn) O(n)
哈希 O(1) O(n)
数组索引 O(1) O(1)
